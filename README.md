# KRDSGallery
CMS Gallery yang dibuat untuk server FiveM KARDES ROLEPLAY
<br>
<p>Cara Install</p>
<ul>
	<li>Clone Git</li>
	<li>Rename /application/config/database.php.bak menjadi database.php . lalu isi dengan detail database</li>
	<li>Import krdsgallery.sql ke database yang diisi di database.php</li>
	<li>Masuk ke admin <domain>/AdministrasiGalleryX/ . Password Default admin:admin (plagiat router)</li>
	</ul>
	<b>DIWAJIBKAN</b> Untuk mengganti password . enkripsi password menggunakan sha512 . enkrip password yang diinginkan dengan sha512 lalu ubah password di database menggunakan phpmyadmin
